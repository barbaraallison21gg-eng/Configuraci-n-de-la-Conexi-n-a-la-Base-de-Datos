# Configuración de Conexión a PostgreSQL 🐘

Este proyecto es una demostración de cómo conectar una aplicación de **Node.js** a una base de datos **PostgreSQL** de forma segura y eficiente utilizando un pool de conexiones.

## 🚀 Tecnologías utilizadas
* **Node.js**: Entorno de ejecución.
* **pg (node-postgres)**: Driver para la comunicación con PostgreSQL.
* **dotenv**: Gestión de variables de entorno para seguridad.
